# simple-object-tracking-using-opencv-template-matching-and-python

## Controls

L - Lock on/off\
W - Make box bigger in y axis\
A - Make box smaller in x axis\
S - Make box smaller in y axis\
D - Make box bigger in x axis\
Q - Quit
